Para rodar o projeto, primeiro baixe o repositório e descompacte ele em uma pasta. 
Em seguida, abra a pasta frontend no cmd e digite "npm i" para baixar os módulos necessários para execução. 
Repita o mesmo para a pasta backend (abra a pasta no cmd e digite 'npm i').
No cmd da pasta frontend, digite "npx serve -l 3000", para inicializar o servidor por parte do cliente
No cmd da pasta backend, digite "node server.js", para inicializar o servidor da api
